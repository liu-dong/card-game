## 实体设计
    玩家：
        属性：持有的牌、玩家类型、玩家名称，进入游戏顺序、计分
        动作：摸牌、出牌、抢地主、不抢、不出、提出牌出完了
    
    牌：
        属性：花色、牌号、序号

    棋牌游戏：
        属性：玩家、一副牌、胜者
        动作：洗牌、发牌、发起抢地主回合、发起出牌回合，判定胜者


## 流程设计

    准备阶段
        开始游戏
        等待玩家加入
        玩家1加入
        玩家2加入
        玩家3加入
        洗牌
        发牌

    游戏阶段
        发起抢地主回合
        问第一个加入游戏的人是否抢地主
            玩家1选择
            玩家2选择
            玩家3选择
        选出地主
    
        地主出牌
        问之后的人是否出牌
            玩家1选择
            玩家2选择
            玩家3选择
        
    结束阶段
        判定胜者
        计分
        游戏结束
